# 步骤 3：设计器代码转换

## 任务目标

将设计器代码文件（.Designer.cs）转换为 WPF XAML 文件（.xaml）

## 执行操作

### 1. 加载转换规则

按照 `prompt/rules/rule_index.md` 中的规则加载流程，加载设计器代码转换规则：
- 读取 `prompt/rules/rule_index.md`
- 根据文件内容识别需要加载的规则文件
- 按优先级加载规则文件

### 2. 执行转换

对每个设计器代码文件进行转换：
- 读取原文件内容
- 生成 XAML 文件到 `{OUTPUT_DIR}\{原文件名}.xaml`
- 按加载的规则进行转换
- 保持 x:Name 与原 Name 一致
- 迁移所有文本属性
- 布局重构和控件替换
