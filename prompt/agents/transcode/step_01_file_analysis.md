# 步骤 1：文件分析

## 任务目标

识别 `{FILES}` 中的文件类型和基本信息，为后续转换做准备。

## 执行操作

1. 列出所有待处理文件
2. 按类型分类：
   - `.Designer.cs` → 设计器代码文件
   - 其他 `.cs` → 后台代码文件
3. 统计文件数量
